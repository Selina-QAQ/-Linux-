# -Linux-
基于 CentOS 8 的运维实战项目，服务器初始化、服务部署、监控系统搭建、自动化脚本开发。

## 技术栈
- 操作系统：CentOS 8
- Web服务：Nginx
- 数据库：MySQL 8
- 监控系统：Zabbix 5.0
- 脚本语言：Shell、Python3

## 项目功能
1. 服务器初始化与安全加固
2. Nginx + MySQL 业务环境部署
3. Zabbix 全栈监控（CPU/内存/磁盘/服务）
4. MySQL 自动备份 Shell 脚本
5. 磁盘使用率监控 Python 脚本
6. 定时任务自动化执行

## 部署步骤
1. 系统初始化（关闭SELinux、配置yum源）
2. 安装 Nginx、MySQL 并配置自启
3. 搭建 Zabbix 监控平台
4. 部署自动化脚本
5. 配置定时任务
